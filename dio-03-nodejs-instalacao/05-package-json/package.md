# Package.json

Criação de modelo de projeto.

    npm init

Perguntas em ordem:

- Nome do Projeto
- Versão
- Descrição
- Arquivo de entrada
- Comando de teste
- Reposistório do GitHub
- Palavras-Chave
- Autor
- Tipo de licença

As configurações principais do projeto inteiro serão salvas dentro do package.json
