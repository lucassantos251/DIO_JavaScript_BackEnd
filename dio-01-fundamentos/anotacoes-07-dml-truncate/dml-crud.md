## Select
É uma consulta que você fará no banco de dados.

### Condições (WHERE / AND / OR)
WHERE (coluna/condição):

- =
- \> / >=
- < / <=
- <> / !=
- LIKE (case-sensitive)
- ILIKE (não case-sensitive)
- IN

Primeira condição sempre **WHERE**.
Demais condiçoes, **AND** ou  **OR**.
